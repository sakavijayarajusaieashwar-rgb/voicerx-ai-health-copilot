# 🚀 VoiceRx – AI Health Copilot

**Voice-first AI healthcare assistant for everyday health management**

---

## 🧠 Overview

VoiceRx is an AI-powered health copilot designed to make healthcare simple, accessible, and conversational.
It helps users manage their health through voice interaction, personalized insights, and smart recommendations.

---

## ❗ Problem

* Elderly users struggle with complex health apps
* Medication management is confusing
* Health advice is generic and hard to understand
* Lack of personalized healthcare guidance

---

## 💡 Solution

VoiceRx provides:

* 🎙️ Voice-first interaction
* 🤖 AI-powered health assistant
* 🧠 Personalized recommendations
* 💬 Simple conversational interface

---

## ⚙️ Features

* Voice-based onboarding
* AI health chat (Mistral-powered)
* Medication tracking & reminders
* Diet & nutrition recommendations
* Medical report analysis (upload + demo sample)
* Emergency-aware responses

---

## 🧑‍⚕️ Target Users

* Elderly individuals
* Chronic disease patients
* Post-surgery recovery users
* Caregivers

---

## 🛠️ Tech Stack

### Frontend

* React + Vite
* Tailwind CSS
* Framer Motion

### Backend

* Node.js + Express
* SQLite

### AI & Voice

* Mistral AI (chat & reasoning)
* Murf AI (text-to-speech)
* Web Speech API (speech-to-text)

---

## 🔐 Environment Variables

Create a `.env` file:

```
VITE_MISTRAL_API_KEY=your_mistral_key
VITE_MURF_API_KEY=your_murf_key
JWT_SECRET=your_secret
```

---

## 🚀 Getting Started

### 1. Clone repo

```
git clone https://github.com/your-username/voicerx-ai-health-copilot.git
cd voicerx-ai-health-copilot
```

---

### 2. Install dependencies

```
npm install
```

---

### 3. Run project

```
npm run dev
```
